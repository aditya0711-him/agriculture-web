# Database - Agriculture Web Application\n\n## Overview\nThis directory contains database schemas, migrations, and seed data.\n\n## Database Schema\n\n### Users Collection/Table\n```\n{\n  id: ObjectId/UUID,\n  username: String,\n  email: String (unique),\n  password: String (hashed),\n  firstName: String,\n  lastName: String,\n  phone: String,\n  location: String,\n  farmSize: Number,\n  cropType: String,\n  experienceLevel: String,\n  createdAt: DateTime,\n  updatedAt: DateTime\n}\n```\n\n### Farms Collection/Table\n```\n{\n  id: ObjectId/UUID,\n  userId: ObjectId/UUID,\n  name: String,\n  location: String,\n  area: Number,\n  soilType: String,\n  waterSource: String,\n  createdAt: DateTime,\n  updatedAt: DateTime\n}\n```\n\n### Crops Collection/Table\n```\n{\n  id: ObjectId/UUID,\n  farmId: ObjectId/UUID,\n  userId: ObjectId/UUID,\n  name: String,\n  variety: String,\n  plantingDate: DateTime,\n  expectedHarvestDate: DateTime,\n  area: Number,\n  quantity: Number,\n  status: String (growing/harvested/failed),\n  health: String (healthy/diseased/stressed),\n  notes: String,\n  createdAt: DateTime,\n  updatedAt: DateTime\n}\n```\n\n### Weather Data Collection/Table\n```\n{\n  id: ObjectId/UUID,\n  farmId: ObjectId/UUID,\n  temperature: Number,\n  humidity: Number,\n  rainfall: Number,\n  windSpeed: Number,\n  date: DateTime,\n  recordedAt: DateTime\n}\n```\n\n### Market Prices Collection/Table\n```\n{\n  id: ObjectId/UUID,\n  cropName: String,\n  price: Number,\n  location: String,\n  unit: String,\n  date: DateTime,\n  source: String,\n  recordedAt: DateTime\n}\n```\n\n## Migrations\n\nDirectory structure for database migrations:\n```\nmigrations/\n├── 001_create_users.sql\n├── 002_create_farms.sql\n├── 003_create_crops.sql\n├── 004_create_weather_data.sql\n└── 005_create_market_prices.sql\n```\n\n## Seeds\n\nInitial seed data for development:\n```\nseeds/\n├── users_seed.json\n├── farms_seed.json\n├── crops_seed.json\n└── market_prices_seed.json\n```\n\n## Database Setup\n\n### MongoDB\n```bash\n# Start MongoDB\nmongod\n\n# Connect to database\nmongo agriculture-web\n```\n\n### PostgreSQL\n```bash\n# Create database\ncreatedb agriculture-web\n\n# Run migrations\npsql agriculture-web < migrations/001_create_users.sql\n```\n\n## Indexes\n\n### MongoDB Indexes\n```javascript\ndb.users.createIndex({ email: 1 }, { unique: true })\ndb.farms.createIndex({ userId: 1 })\ndb.crops.createIndex({ farmId: 1 })\ndb.crops.createIndex({ userId: 1 })\n```\n\n## Backup & Recovery\n\n### Backup\n```bash\n# MongoDB\nmongodump --db agriculture-web --out ./backup\n\n# PostgreSQL\npg_dump agriculture-web > backup.sql\n```\n\n### Restore\n```bash\n# MongoDB\nmongorestore ./backup\n\n# PostgreSQL\npsql agriculture-web < backup.sql\n```\n\n## Contributing\nSee CONTRIBUTING.md in the root directory.\n"
